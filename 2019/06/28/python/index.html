在Ubuntu中安装python中所遇到的一些常见问题以及解决方案

因为Ubuntu系统中不同于window，初次使 用Ubuntu难免遇到各种各种的问题

①提示各种没有权限

提示各种没有权限

解决方法：在每条命令行前加入“sudo”即使用最高权限来执行命令，输入账号的密码后即可执行

② linux系统安装软件是有时会碰到“dpkg 被中断,您必须手工运行 sudo dpkg –configure -a解决此问题”，然而按照提示运行却并没能很好的解决问题。其实导致这个问题的主要原因是因为/var/lib/dpkg/updates文件下的文件有问题，可能是其他软件安装过程或是其他原因导致的，这里删除掉然后重建即可。

解释：[]sudo apt-get update指令会重新建立这些资料，所以不必担心删除后会出问题；
[]sudo apt-get upgrade会更新你的电脑里面已安装的软件的明细，根据软件的明细更新软件到最新版。
PS：“dpkg ”是“Debian Packager ”的简写。为 “Debian” 专门开发的套件管理系统，方便软件的安装、更新及移除。所有源自“Debian”的“Linux ”发行版都使用 “dpkg”另外该行为是删除所有update文件并且重新下载更新，可能需要比较多的时间，请耐心等待。
